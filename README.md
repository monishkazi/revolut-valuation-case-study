# revolut-valuation-case-study
Objective - To evaluate whether Revolut’s ~$75bn valuation is justified given its growth, profitability, and macro conditions. 
Methodology- EV/Revenue Comparable Analysis Two-Stage Discounted Cash Flow (DCF) Model VC/IPO Back-Solve Valuation.
Key Insights -
Revolut trades at ~3x peer median valuation
DCF suggests overvaluation under conservative assumptions
Valuation justified only under high-growth execution scenarios
Conclusion-
Revolut is fairly valued with an upward bias, contingent on successful execution of growth and monetisation strategy.
